개인정보처리방침

최종 업데이트: 2024년 10월 15일

1. 소개

MathStep("앱")은 수학 학습을 지원하도록 설계된 교육 앱입니다.

저희는 귀하의 개인정보를 존중하며 보호에 최선을 다하고 있습니다.

이 개인정보처리방침은 앱을 사용할 때 수집하는 정보에 대해 설명합니다.

2. 수집하는 정보

다음 정보를 수집합니다:

  • 입력한 수학 표현식 (교육 목적)
  • 앱 사용 데이터 (크래시 리포트, 성능 데이터)
  • 기기 정보 (OS, 앱 버전, 기기 모델)
  • 광고 관련 정보 (AdMob을 통해)

2.1 AdMob 데이터 수집

이 앱은 Google AdMob을 사용하여 광고를 표시합니다. AdMob은 다음 정보를 수집할 수 있습니다:

  • 기기 정보: 기기 ID, 광고 ID, 기기 유형, OS, 언어 설정
  • 앱 정보: 앱 버전, 패키지명, 설치 정보
  • 광고 관련 정보: 광고 노출 횟수, 클릭 수, 광고 효과 측정 데이터
  • 네트워크 정보: IP 주소 (익명화), 네트워크 연결 유형
  • 위치 정보: 대략적인 위치 정보 (국가/지역 수준, 기기 설정으로 제어 가능)
  • 사용 데이터: 앱 사용 시간, 기능 사용 통계

참고: AdMob은 개인 식별 정보(PII)를 수집하지 않습니다. 수집되는 데이터는 익명화되거나 의사 익명화됩니다.

3. 수집하지 않는 정보

다음 정보는 수집하지 않습니다:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 위치 데이터
  • 연락처 정보
  • 카메라/마이크의 오디오 또는 이미지 데이터 (현재 앱에서는 이러한 권한을 사용하지 않음)

4. 정보 사용 방법

수집한 정보는 다음 목적으로 사용합니다:

  • 수학 문제 해결 지원 제공
  • 앱 기능 개선
  • 오류 및 버그 수정
  • 광고 표시 (AdMob)

4.1 AdMob 데이터 사용 목적

AdMob이 수집한 데이터는 다음 목적으로 사용됩니다:

  • 광고 배송: 사용자에게 관련성 높은 광고를 표시하기 위해
  • 광고 성과 측정: 광고 노출 횟수, 클릭 수, 전환율 측정
  • 광고 최적화: 광고 품질과 관련성 향상을 위해
  • 사기 방지: 광고의 부정 클릭이나 부정 노출 탐지 및 방지
  • 리포트 작성: 광고주를 위한 리포트 작성 (개인을 식별할 수 없는 형태로)
  • 법적 요구사항 준수: 관련 법률 및 규정 준수

5. 제3자 공유

다음 경우를 제외하고는 제3자와 정보를 공유하지 않습니다:

  • OpenAI API (수학 표현식 분석용)
  • Google AdMob (광고 표시용)
  • 법률에 의해 요구되는 경우

6. 데이터 저장

  • 수학 표현식은 기기에 로컬 저장됩니다
  • 수학 표현식만 인터넷을 통해 OpenAI API로 전송됩니다
  • 앱 내에서 언제든지 기록을 삭제할 수 있습니다
  • 로컬 데이터(수학 표현식, 앱 설정)는 앱을 제거하거나 앱 내 기록 삭제 기능을 사용하여 삭제할 수 있습니다

6.1 AdMob 데이터 저장

  • 광고 ID: 기기에 로컬 저장되며 재설정 가능
  • 광고 데이터: Google 서버에 최대 24개월간 저장
  • 분석 데이터: 익명화된 형태로 최대 26개월간 저장
  • 위치 정보: 대략적인 위치 정보만 저장 (상세한 위치는 저장되지 않음)

6.2 사용자 선택권

AdMob 데이터 수집에 대해 다음 선택권이 있습니다:

  • 광고 ID 재설정: 기기 설정에서 광고 ID를 재설정할 수 있습니다
  • 개인화 광고 비활성화: 기기 설정에서 개인화 광고를 비활성화할 수 있습니다
  • 위치 접근 제한: 앱의 위치 접근을 제한할 수 있습니다
  • 데이터 삭제: Google 계정 설정에서 AdMob 데이터 삭제를 요청할 수 있습니다
  • 데이터 수집 완전 중단: AdMob 데이터 수집을 완전히 중단하려면 앱을 제거하세요

참고: 이러한 설정을 변경해도 광고 표시는 계속되지만, 관련성이 낮은 광고가 표시될 수 있습니다.

7. 아동 개인정보

저희 앱은 13세 미만 아동에게도 안전합니다

COPPA(아동 온라인 개인정보보호법)를 준수합니다

아동으로부터 개인정보를 수집하지 않습니다

7.1 AdMob과 아동 개인정보

  • AdMob은 COPPA를 준수하며 13세 미만 아동으로부터 개인정보를 수집하지 않습니다
  • 아동에게는 연령에 적합한 광고만 표시됩니다
  • 개인화 광고는 13세 미만 사용자에게 표시되지 않습니다
  • 13세 미만 사용자에게는 연령에 적합한 광고만 자동으로 표시됩니다

8. 개인정보처리방침 변경

이 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다

중요한 변경사항은 앱 내에서 알려드립니다

9. 국제 배포

유럽 경제 지역(EEA) 및 영국 사용자의 경우, 개인화 광고에 대해 추가 동의 관리가 적용될 수 있습니다.
